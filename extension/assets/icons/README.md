# Extension icon assets

Place PNG icons here following Chrome's naming convention:

| File | Size |
|------|------|
| `icon16.png` | 16 × 16 px |
| `icon32.png` | 32 × 32 px |
| `icon48.png` | 48 × 48 px |
| `icon128.png` | 128 × 128 px |

Reference them in `manifest.json` under the `"icons"` key and in `"action.default_icon"`.
